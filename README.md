# Portfolio de Maximiliano Arrieta

Portfolio personal creado para la actividad integradora de Desarrollo de Web, enfocado en presentar mi marca personal y proyectos destacados.

## Tecnologías Utilizadas
* HTML5
* CSS3 (Estilos personalizados)
* Bootstrap 5.3 (Framework CSS mediante CDN)

## Visualización Local
Para visualizar este proyecto de forma local:
1. Clona o descarga este repositorio en tu computadora.
2. Abre la carpeta del proyecto.
3. Haz doble clic en el archivo `index.html` para abrirlo en tu navegador web predeterminado.
