🍽️ Sistema de Ventas — Restaurante
🍽️

Sistema completo de gestión de ventas para un restaurante, desarrollado con ASP.NET y SQL Server. Permite administrar pedidos, productos, mesas y generar reportes de ventas. El proyecto incluye autenticación de usuarios y diferentes roles de acceso.

Tecnologías

ASP.NET C# SQL Server HTML/CSS

Características

  • Gestión de pedidos y mesas en tiempo real
  • Catálogo de productos con precios
  • Reportes de ventas diarias y mensuales
  • Sistema de autenticación con roles
🗄️ Sistema de Administración de BD

Sistema de administración de bases de datos con arquitectura separada en capas (presentación, negocio y datos) y una WebApi REST que permite consumirlo desde cualquier tecnología cliente. Diseñado para máxima escalabilidad y reutilización.

Tecnologías

ASP.NET MVC WebApi REST Entity Framework SQL Server C#

Arquitectura

  • Separación en capas: UI, Negocio, Datos
  • WebApi REST para integración con clientes externos
  • Patrón repositorio con Entity Framework
  • CRUD completo sobre múltiples entidades
🚗 Auto RC por WiFi — ESP-32
🚗

Proyecto de hardware e IoT: un auto a control remoto controlado por WiFi utilizando un ESP-32. La comunicación se realiza mediante el protocolo MQTT, con un dashboard visual en NODE-RED y registro de telemetría en MySQL. Combina electrónica, firmware y software.

Tecnologías

ESP-32 MQTT NODE-RED MySQL WiFi

Características

  • Control en tiempo real vía WiFi desde cualquier dispositivo
  • Dashboard de control con NODE-RED
  • Registro de movimientos y telemetría en MySQL
  • Protocolo MQTT para mensajería liviana y eficiente
